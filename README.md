# 🚀 Smart Report Platform (RAG-Driven)

منصة ذكية متقدمة تعتمد على تقنيات **RAG (Retrieval-Augmented Generation)** لتحليل البيانات ورفع التقارير. تم تطوير هذا المشروع كجزء من بحث الماجستير الخاص بـ **رشاد يوسف خرما**، ويستخدم الذكاء الاصطناعي لتحويل الملفات الخام إلى تقارير تفاعلية احترافية.

---

## ✨ المميزات الرئيسية
* **التحليل الذكي:** استخدام نموذج `Llama-3.3-70b-versatile` لتحليل البيانات والنصوص بدقة عالية.
* **دعم ملفات متعددة:** معالجة ملفات Excel (xlsx), CSV وملفات PDF.
* **نظام الدردشة (Context-Aware Chat):** مساعد ذكي مدمج يسمح للمستخدم بمناقشة محتوى التقرير المولد فقط.
* **توليد الرسوم البيانية:** عرض مرئي تلقائي للبيانات الرقمية باستخدام `Chart.js`.
* **الوضع الليلي (Dark Mode):** واجهة مستخدم عصرية تدعم التبديل بين الأنماط البصرية.

---

## 🛠️ المتطلبات التقنية (Prerequisites)
تأكد من تثبيت Python 3.9 أو أحدث على جهازك.

المكتبات المطلوبة:
* **Flask:** لإدارة السيرفر والواجهة البرمجية.
* **Pandas & NumPy:** لمعالجة البيانات الرقمية والجداول.
* **Groq SDK:** للاتصال بنموذج الذكاء الاصطناعي.
* **PyMuPDF (fitz):** لاستخراج النصوص من ملفات PDF.

---

## ⚙️ كيفية التشغيل محلياً (Step-by-Step)

اتبع الخطوات التالية بدقة لتشغيل المشروع:

### 1. تحميل المشروع
قم بتحميل ملفات المشروع من المستودع:
```bash
git clone [https://github.com/YOUR_USERNAME/Smart-Report-Platform.git](https://github.com/YOUR_USERNAME/Smart-Report-Platform.git)
cd Smart-Report-Platform
2. تثبيت المكتبات (Dependencies)
افتح واجهة الأوامر (Terminal) في مجلد المشروع ونفذ الأمر التالي:
pip install flask pandas numpy groq openpyxl pymupdf
3. تشغيل السيرفر
نفذ الأمر التالي لبدء تشغيل المنصة:
python app.py
